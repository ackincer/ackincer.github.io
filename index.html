Web Dev Fundamentals Cheat Sheet

Starting an HTML Document

Requirement Code
Doc Type <!DOCTYPE html>
CSS Link <link href="link to local css stylesheet" type="text/css" rel="stylesheet" />
Common Tags <html>, <head>, <body>, <footer>

CSS Selectors

Selector Meaning Example
Universal Applies to all elements in the document * {}
Type Matches element names h1, h2, h3 {}
Class Matches an element's class value .note {}, p.note {}
ID Matches an element's ID value #introduction {}
Child Matches an element that is a direct child of another p>a {}
Descendant Matches an element that is a descendant of another element, not only a direct child p a {}

CSS Declarations and Values

Declaration Meaning Example
opacity Specify the opacity of an element with a value between 0 (0% opacity) and 1 (100% opacity) opacity: 0.5;
hsla Hue (an angle between 0 and 360 degrees), saturation (percentage), lightness (percentage, with 0% being white and 100% being black), and alpha (or opacity, between 0 and 1.0) background-color: { hsla(0, 100%, 100%, 0.5); }
@font-face Use a font, even if it isn't downloaded on the user's computer @font-face { font-family: 'Name of font; src: url('link to fonts by provider, and requires links for eot, woff, etc'); }
line-height Increase or decrease the vertical space between lines of text using ems, which are relative to the text size (1.4 is standard) p { line-height: 1.4em;
letter-spacing Manage the space between letters p { letter-spacing: 02em; }
padding Add space between the text and the edges of the screen, first % is top and bottom, second is left and right body { padding: 5% 10%; }

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<th> Table Heading The table row title.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
